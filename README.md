# FC2203S 有刷电机半桥驱动

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

基于 FC2203S 芯片的有刷电机半桥驱动开源硬件项目，适用于气泵、小型电机等应用场景。

## 项目简介

本项目开源了完整的电机驱动设计，包括：

- **嘉立创工程文件**：可直接导入嘉立创 EDA 进行编辑与打样
- **原理图与 PCB PDF**：便于查看与学习
- **测试数据**：上管开启时间等实测波形

所有元器件均可在淘宝等平台购买，**整体复刻成本约 5 元**，适合 DIY 与学习。

## 测试数据

- **上管开启时间**：使用 10Ω 栅极限流电阻时约 **400ns**
- **更换 NMOS 时**：需根据所选 MOS 管的参数调整栅极限流电阻阻值

## 使用说明

1. 使用嘉立创 EDA 打开 `嘉立创源文件/气泵驱动测试V1.epro2`
2. 根据 BOM 采购元器件（可参考 PDF 中的元件清单）
3. 打样 PCB 并焊接
4. 若更换不同型号 NMOS，请相应调整栅极限流电阻
